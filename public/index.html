Complaint Management System

Manages your complaints Efficiently

General Guidelines

  • Do not submit fake or inappropriate complaints.
  • Follow system policies.
  • Respect response time.
  • Contact admin only if urgent.

Before Submitting Complaint

  • Provide correct personal details.
  • Select appropriate complaint category.
  • Write clear and detailed description.
  • Avoid submitting duplicate complaints.

After Submitting complaint make sure that

  • Note your Complaint ID carefully.
  • Use Complaint ID to track status.
  • Wait for admin to assign staff.
  • You will be notified once resolved.
